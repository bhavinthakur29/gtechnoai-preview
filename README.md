# G-Techno AI - Preview

A high-performance, fully responsive React landing page demo designed for **G-Techno AI**. This preview showcases a modern UI with premium entrance animations to establish an award-winning digital presence before final production approval.

## 🚀 Features

*   **Premium Animations:** Built with Framer Motion for smooth, staggered entrance effects.
*   **Fully Responsive:** Fluid typography and adaptive layouts that look perfect on mobile, tablet, and desktop.
*   **Modern UI/UX:** Dark-themed aesthetic with ambient background glows and glassmorphism elements.
*   **Performance Optimized:** Lightweight component structure powered by Vite.

## 🛠️ Tech Stack

*   [React](https://reactjs.org/) (via [Vite](https://vitejs.dev/))
*   [Tailwind CSS](https://tailwindcss.com/) - Styling & responsiveness
*   [Framer Motion](https://www.framer.com/motion/) - UI animations
*   [Lucide React](https://lucide.dev/) - Iconography

## 💻 Local Development

To run this preview locally on your machine:

**1. Clone the repository**
\`\`\`bash
git clone https://github.com/YOUR_USERNAME/gtechnoai-preview.git
cd gtechnoai-preview
\`\`\`

**2. Install dependencies**
Make sure you have Node.js installed, then run:
\`\`\`bash
npm install
\`\`\`
*(Note: This project requires `framer-motion` and `lucide-react`)*

**3. Start the development server**
\`\`\`bash
npm run dev
\`\`\`
